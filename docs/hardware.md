## жіЁж„ҸдәӢйЎ№

### з”өжәҗ
1. **еҝ…йЎ»е…ұең°**: жүҖжңүи®ҫеӨҮзҡ„ең°зәҝеҝ…йЎ»иҝһжҺҘеңЁдёҖиө·
2. **з”өжңәйҡ”зҰ»**: L298N зҡ„з”өжңәз”өжәҗе’ҢйҖ»иҫ‘з”өжәҗеҲҶејҖ
3. **ж»Өжіўз”өе®№**: еңЁз”өжңәз”өжәҗеӨ„ж·»еҠ  100ОјF з”өе®№
4. **дҝқйҷ©дёқ**: еңЁз”өжұ жӯЈжһҒж·»еҠ  5A дҝқйҷ©дёқ

### дҝЎеҸ·
1. **иҲөжңәдҝЎеҸ·**: дҪҝз”Ё PWM дҝЎеҸ·пјҢйў‘зҺҮ 50Hz
2. **з”өжңә PWM**: йў‘зҺҮ 1kHz-20kHz
3. **ж— зәҝеӨ©зәҝ**: ESP32 зҡ„еӨ©зәҝеҢәеҹҹйҒҝе…ҚйҮ‘еұһйҒ®жҢЎ
4. **ж‘„еғҸеӨҙжҺ’зәҝ**: OV2640 зҡ„жҺ’зәҝе°ҪйҮҸзҹӯпјҢйҒҝе…Қе№Іжү°

### е®үе…Ё
1. **зҙ§жҖҘеҒңжӯў**: дҝқз•ҷзү©зҗҶжҖҘеҒңжҢүй’®
2. **з”өжәҗејҖе…і**: ж·»еҠ жҖ»з”өжәҗејҖе…і
3. **иҝҮзғӯдҝқжҠӨ**: L298N ж·»еҠ ж•ЈзғӯзүҮ
4. **еҸҚжҺҘдҝқжҠӨ**: з”өжұ жҺҘеҸЈж·»еҠ еҸҚжҺҘдҝқжҠӨдәҢжһҒз®Ў
